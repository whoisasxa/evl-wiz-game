Features

Four playable classes:

Warrior – heavy melee + shield block

Mage – high burst damage + mana shield

Archer – quick multi-attacks + full evasion

Paladin – holy smite + divine shield + minor heal

Enemy: Evil Wizard with automatic health regeneration

Randomized attack damage

Two special abilities per class

Healing mechanic with max-health limits

Full turn-based battle loop

Win or defeat conditions with end-game messages

OOP Concepts Used:

Classes & Inheritance

Method Overriding

Encapsulation (attributes controlling behavior)

Interactions between objects

State effects (evade, shield, health regen)


How to Play

Run the game:

python evl_wiz.py


Select your character class.

Each turn choose:

Attack

Use Special Ability

Heal

View Stats

Survive the Wizard’s regeneration + attack cycle.

Defeat The Dark Wizard to win!
